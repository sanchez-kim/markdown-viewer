## 기호 세 개로 가로줄 긋기

구분선(수평선, horizontal rule)은 한 줄에 다음 기호 중 하나를 **세 개 이상** 적으면 됩니다.

```
---
***
___
```

셋 다 똑같은 가로줄로 렌더됩니다. 보통 가장 많이 쓰는 건 하이픈 세 개(`---`)입니다. 긴 문서에서 장(章)을 나누거나, 본문과 맺음말을 구분할 때 유용합니다.

## ⚠️ 흔한 실수: 제목으로 변해버림

`---`를 **글자 바로 아랫줄**에 붙여 쓰면 구분선이 아니라 **그 윗줄을 제목(H2)으로** 만들어 버립니다. 이는 마크다운의 또 다른 제목 문법이기 때문입니다.

```
중요한 문장
---
```

위 코드는 "중요한 문장"을 큰 제목으로 바꿉니다. 구분선을 의도했다면 **위에 빈 줄을 하나** 넣으세요.

```
중요한 문장

---
```

이렇게 하면 의도대로 문장과 구분선이 따로 나옵니다.

정리하면, 구분선은 `---`·`***`·`___`를 세 개 이상 쓰되, **앞 줄과 빈 줄로 띄우는 것**만 기억하면 됩니다.
