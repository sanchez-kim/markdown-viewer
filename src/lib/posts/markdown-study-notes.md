## 마크다운으로 필기하면 좋은 이유

강의 노트를 마크다운으로 정리하면 세 가지 장점이 있습니다. 첫째, 제목과 목록으로 **내용의 구조**가 한눈에 보입니다. 둘째, 텍스트 기반이라 **검색**이 쉽습니다. 셋째, 어떤 기기에서도 열리고, PDF로 변환해 출력하거나 공유하기도 편합니다.

## 강의 노트 기본 구조

```
# 자료구조 - 3주차: 스택과 큐

## 핵심 개념
- **스택(Stack)**: LIFO (Last In First Out)
- **큐(Queue)**: FIFO (First In First Out)

## 상세 정리
### 스택의 주요 연산
- push: 데이터 추가
- pop: 데이터 제거

## 외울 것 (시험 대비)
- [ ] 스택/큐 시간복잡도
- [ ] 활용 사례 3가지

## 질문 / 헷갈리는 점
- 큐와 덱(deque)의 차이?
```

## 학생에게 특히 유용한 문법

-   **제목 계층**: 과목 → 주차 → 개념 순으로 `#`, `##`, `###`를 쓰면 구조가 명확해집니다.
-   **체크리스트**: 외울 항목, 과제 목록을 `- [ ]`로 관리하면 진도 체크가 됩니다.
-   **표**: 비교가 필요한 개념(예: 정렬 알고리즘별 복잡도)은 표로 정리하면 한눈에 들어옵니다.
-   **인용구**: 교수님이 강조한 내용이나 핵심 정의를 `>`로 강조해 둡니다.
-   **코드 블록**: 프로그래밍 과목이라면 예제 코드를 문법 강조와 함께 보관합니다.

## 복습 효율을 높이는 팁

-   **"질문" 섹션 만들기**: 헷갈리는 점을 따로 적어두고, 해결하면 지웁니다.
-   **나만의 요약**: 강의 내용을 그대로 옮기지 말고, 내 말로 다시 정리하면 기억에 오래 남습니다.
-   **과목별 파일 분리**: 한 파일에 다 넣지 말고 과목·주차별로 나누면 찾기 쉽습니다.

## 시험 기간 활용

평소 마크다운으로 정리해두면, 시험 직전에 "외울 것" 체크리스트와 표만 빠르게 훑어볼 수 있습니다. 필요하면 PDF로 내보내 인쇄하거나 태블릿에서 보세요. 잘 구조화된 노트 한 장이 두꺼운 교재보다 효율적일 때가 많습니다.

## 정리

마크다운 필기는 처음엔 조금 낯설어도, 익숙해지면 손글씨보다 빠르고 정리도 깔끔합니다. 이지 마크다운에서 강의 노트를 작성하고 PDF로 내보내, 나만의 공부 자료를 만들어보세요.
