## 마크다운이란?

마크다운(Markdown)은 2004년 존 그루버(John Gruber)가 만든 경량 마크업 언어입니다. HTML처럼 복잡한 태그 없이, 일반 텍스트에 특수 기호를 더해 서식을 표현합니다. 작성한 텍스트가 그대로 읽히면서도 변환하면 깔끔한 HTML이 됩니다.

오늘날 GitHub, Notion, Slack, Reddit, Discord 등 수많은 플랫폼이 마크다운을 지원합니다. 개발자뿐 아니라 작가, 학생, 기획자 등 다양한 사람들이 문서 작성에 활용하고 있습니다.

## 제목 (Headings)

제목은 `#` 기호로 만듭니다. `#`의 개수가 많을수록 작은 제목이 됩니다.

```
# 제목 1 (H1)
## 제목 2 (H2)
### 제목 3 (H3)
#### 제목 4 (H4)
```

일반적으로 H1은 문서 전체 제목에 한 번만 사용하고, 본문 구조는 H2부터 시작하는 것이 좋습니다. 제목 계층은 SEO에도 중요하게 작용합니다.

## 텍스트 강조

텍스트를 굵게, 기울게, 취소선 등으로 강조할 수 있습니다.

```
**굵게** 또는 __굵게__
*기울임* 또는 _기울임_
~~취소선~~
**굵게 _굵게+기울임_ 굵게**
```

결과: **굵게**, *기울임*, ~취소선~

강조는 의미 있는 곳에만 사용하세요. 너무 많이 쓰면 정작 강조가 필요한 부분이 묻혀버립니다.

## 목록 (Lists)

순서 없는 목록은 `-`, `*`, `+` 중 하나로 시작합니다.

```
- 사과
- 바나나
  - 덜 익은 바나나
  - 잘 익은 바나나
- 포도
```

순서 있는 목록은 숫자로 시작합니다. 실제 숫자가 달라도 순서대로 렌더링됩니다.

```
1. 첫 번째
2. 두 번째
3. 세 번째
```

목록을 2칸 들여쓰면 중첩 목록을 만들 수 있습니다. 복잡한 계층 구조를 표현할 때 유용합니다.

## 링크와 이미지

링크는 대괄호 안에 텍스트, 소괄호 안에 URL을 씁니다.

```
[이지 마크다운](https://easy-md.com)
[이메일 보내기](mailto:example@email.com)
```

이미지는 링크 문법 앞에 `!`를 붙입니다. 대괄호 안은 대체 텍스트(alt text)입니다.

```
![로고 이미지](https://example.com/logo.png)
![로고](./images/logo.png "마우스 올리면 나오는 설명")
```

대체 텍스트는 이미지가 로드되지 않을 때 표시되고, 스크린 리더가 읽어주므로 접근성에 중요합니다.

## 코드

인라인 코드는 백틱(`` ` ``) 하나로 감쌉니다. 코드 블록은 백틱 세 개로 열고 닫습니다.

````
이것은 `인라인 코드`입니다.

```javascript
function hello(name) {
  console.log(`안녕하세요, ${name}님!`);
}
hello('세계');
```
````

코드 블록 시작 부분에 언어 이름을 적으면 문법 강조(syntax highlighting)가 적용됩니다. `javascript`, `python`, `bash`, `sql`, `css` 등 수백 가지 언어를 지원합니다.

## 인용구 (Blockquote)

`>` 기호로 인용구를 만듭니다. 중첩도 가능합니다.

```
> 명언을 인용할 때 사용합니다.
> 여러 줄도 가능합니다.
>
>> 중첩 인용구
```

인용구는 참고 문헌, 중요 메시지, 주의사항을 강조할 때 효과적입니다.

## 표 (Table)

표는 파이프(`|`)와 하이픈(`-`)으로 만듭니다.

```
| 이름 | 나이 | 직업 |
|------|------|------|
| 홍길동 | 30 | 개발자 |
| 김철수 | 25 | 디자이너 |
```

열 정렬은 두 번째 행에서 콜론(`:`) 위치로 지정합니다.

```
| 왼쪽 | 가운데 | 오른쪽 |
|:-----|:------:|------:|
| 텍스트 | 텍스트 | 텍스트 |
```

## 체크리스트

체크리스트는 GitHub, Notion 등에서 태스크 관리에 많이 씁니다.

```
- [x] 완료된 항목
- [ ] 미완료 항목
- [ ] 또 다른 할 일
```

## 구분선

세 개 이상의 하이픈, 별표, 밑줄로 수평선을 만듭니다.

```
---
***
___
```

## 이스케이프

마크다운 특수 문자를 그대로 출력하려면 백슬래시(`\`)를 앞에 붙입니다.

```
\*별표를 그대로 보여주기\*
\# 제목 기호 그대로 보여주기
```

## 실전 팁

-   **일관성 유지**: 같은 문서에서 `-`와 `*`를 섞어 쓰지 마세요.
-   **빈 줄 활용**: 단락 사이, 제목 앞뒤에 빈 줄을 넣으면 가독성이 올라갑니다.
-   **미리보기 활용**: 작성 중에 미리보기를 켜두면 실수를 줄일 수 있습니다.
-   **상대 경로**: 이미지는 절대 경로보다 상대 경로를 쓰면 이식성이 좋습니다.

마크다운은 배우는 데 30분이면 충분하지만, 잘 활용하면 문서 작성 속도가 2배 이상 빨라집니다. 이지 마크다운에서 직접 연습해보세요.
