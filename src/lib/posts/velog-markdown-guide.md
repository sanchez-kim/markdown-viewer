## Velog는 마크다운 네이티브

Velog(벨로그)는 개발자를 위한 블로그 서비스로, 글쓰기 자체가 마크다운 기반입니다. 별도의 모드 전환 없이 처음부터 마크다운으로 작성하고, 오른쪽에 실시간 미리보기가 함께 표시됩니다. 마크다운에 익숙한 개발자라면 가장 편하게 쓸 수 있는 플랫폼 중 하나입니다.

## 코드 블록 강조

Velog는 코드 블록의 문법 강조를 깔끔하게 지원합니다. 언어를 지정하면 그에 맞게 색이 입혀집니다.

````
```typescript
interface User {
  id: number;
  name: string;
}
```
````

코드 블록 상단에 파일명을 표시하고 싶다면, 언어 뒤에 콜론과 파일명을 붙이는 Velog 확장 문법(`ts:user.ts`)을 쓸 수 있습니다.

## 이미지 업로드

이미지는 본문에 파일을 끌어다 놓거나 클립보드에서 바로 붙여넣으면 자동으로 업로드되고 마크다운 문법이 삽입됩니다. 스크린샷을 캡처해 바로 Ctrl+V 하는 방식이 매우 빠릅니다.

## 목차(Table of Contents)

Velog는 제목(`#`, `##`, `###`) 구조를 분석해 글 우측에 목차를 자동으로 생성합니다. 따라서 제목 계층을 논리적으로 잘 잡으면, 별도 작업 없이 깔끔한 목차가 만들어집니다. H2와 H3를 적절히 섞어 구조를 만드세요.

## 시리즈 기능

여러 글을 묶어 시리즈로 만들 수 있습니다. 연재형 튜토리얼이나 학습 기록을 정리할 때 유용합니다. 글 발행 시 시리즈에 추가하면 독자가 순서대로 읽어나갈 수 있습니다.

## 작성 팁

-   **썸네일 설정**: 발행 화면에서 대표 이미지를 지정하면 공유 시 더 눈에 띕니다.
-   **태그 활용**: 관련 태그를 달면 같은 주제를 찾는 독자에게 노출됩니다.
-   **원본 백업**: Velog 글도 마크다운 원문을 따로 저장해두면 다른 곳에 재활용하기 좋습니다.

## 정리

Velog는 마크다운을 가장 자연스럽게 쓸 수 있는 개발 블로그 플랫폼입니다. 외부 에디터에서 초안을 다듬고 Velog에 옮기면, 글의 완성도와 작성 속도를 모두 잡을 수 있습니다. 마크다운 문법에 익숙해질수록 글쓰기가 즐거워집니다.
