## 엔터만으로는 줄바꿈이 안 됩니다

마크다운에서 한 번 엔터(줄바꿈)를 치면, 결과 화면에서는 **줄이 바뀌지 않고 같은 줄로 이어집니다**. 마크다운이 한 줄 띄우기를 무시하기 때문입니다. 줄바꿈을 하려면 다음 세 가지 중 하나를 써야 합니다.

## 방법 1. 줄 끝에 공백 두 칸

줄 맨 끝에 **스페이스를 두 번** 누르고 엔터를 치면 줄이 바뀝니다.

```
첫 번째 줄··
두 번째 줄
```

(`··`는 공백 두 칸을 표시한 것입니다.) 가장 표준적인 방법이지만, 눈에 보이지 않아 실수하기 쉽습니다.

## 방법 2. 백슬래시(\)

줄 끝에 백슬래시 하나를 붙이는 방법입니다. 공백과 달리 **눈에 보여서** 편집할 때 헷갈리지 않습니다.

```
첫 번째 줄\
두 번째 줄
```

다만 일부 구형 에디터에서는 지원하지 않을 수 있습니다.

## 방법 3. `<br>` 태그

HTML을 허용하는 곳이라면 `<br>` 태그로 확실하게 줄을 바꿀 수 있습니다.

```
첫 번째 줄<br>
두 번째 줄
```

## 줄바꿈과 문단 나누기는 다릅니다

위 방법들은 **같은 문단 안에서 줄만 바꿉니다**. 완전히 새로운 문단(위아래 여백 포함)을 만들려면, 두 줄 사이를 **빈 줄 하나**로 띄우면 됩니다.

```
첫 번째 문단입니다.

두 번째 문단입니다.
```

정리하면, 줄만 바꾸려면 공백 두 칸·`\`·`<br>`, 문단을 나누려면 빈 줄을 쓰면 됩니다. 이지 마크다운에서는 입력과 동시에 결과가 보이므로 직접 눌러보며 차이를 확인해보세요.
