## 기본 링크 문법

링크는 대괄호 안에 보일 텍스트, 소괄호 안에 주소를 적습니다.

```
[이지 마크다운](https://easy-md.com)
```

제목(title)을 추가하면 마우스를 올렸을 때 설명이 뜹니다.

```
[이지 마크다운](https://easy-md.com "무료 마크다운 에디터")
```

이메일이나 전화 링크도 만들 수 있습니다: `[메일 보내기](mailto:hello@example.com)`

## 참조 링크(Reference Link)

같은 링크를 여러 번 쓰거나 본문을 깔끔하게 유지하고 싶을 때는 참조 링크가 유용합니다. 주소를 문서 하단에 따로 모아두는 방식입니다.

```
자세한 내용은 [공식 문서][docs]를 참고하세요.
설치는 [가이드][docs]에 있습니다.

[docs]: https://example.com/docs
```

본문에서는 이름표만 쓰고 실제 주소는 한 곳에서 관리하므로, 주소가 바뀌어도 한 줄만 고치면 됩니다.

## 이미지 삽입

이미지는 링크 문법 앞에 느낌표(`!`)를 붙입니다. 대괄호 안은 대체 텍스트(alt text)입니다.

```
![프로젝트 로고](./images/logo.png)
![스크린샷](https://example.com/shot.png "메인 화면")
```

## alt 텍스트는 왜 중요한가

대체 텍스트는 이미지가 로드되지 않을 때 대신 표시되고, 시각장애인용 스크린 리더가 읽어줍니다. 검색 엔진도 이미지를 이해하는 단서로 사용하므로 SEO에도 도움이 됩니다. "이미지", "사진" 같은 무의미한 단어 대신, 이미지 내용을 구체적으로 적으세요.

## 이미지 크기 조절

표준 마크다운은 이미지 크기 조절 문법이 없습니다. 크기를 지정하려면 HTML 태그를 직접 쓰면 됩니다(대부분의 마크다운 도구가 HTML을 허용합니다).

```
<img src="./logo.png" alt="로고" width="200">
```

## 링크를 이미지로 감싸기

이미지를 클릭하면 다른 페이지로 이동하게 하려면, 이미지 문법을 링크 문법으로 한 번 더 감싸면 됩니다.

```
[![로고](./logo.png)](https://example.com)
```

README 상단의 뱃지나 배너 이미지에 자주 쓰이는 패턴입니다.
