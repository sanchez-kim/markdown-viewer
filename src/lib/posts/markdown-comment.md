## 마크다운에는 주석 문법이 없습니다

코드처럼 "화면엔 안 보이고 원문에만 남는 메모"를 마크다운 자체 문법으로는 만들 수 없습니다. 대신 **HTML 주석**을 쓰면 됩니다.

## HTML 주석 `<!-- -->`

`<!--`와 `-->` 사이의 내용은 렌더링되지 않습니다. HTML을 처리하는 대부분의 마크다운 환경에서 동작합니다.

```
<!-- 이 줄은 화면에 안 보입니다. 작성용 메모. -->

본문은 정상적으로 보입니다.
```

작성하다가 "여기 나중에 보강" 같은 메모를 남기거나, 일부 문단을 잠시 가려 둘 때 유용합니다.

```
<!--
임시로 숨겨둔 문단.
나중에 다시 살릴 예정.
-->
```

## HTML이 막힌 곳을 위한 대안

드물게 HTML 주석조차 그대로 노출되는 환경이 있습니다. 이럴 때 쓰는 우회법이 링크 정의를 이용한 트릭입니다.

```
[//]: # (이 줄도 화면에 보이지 않습니다)
```

쓰이지 않는 링크 라벨이라 어디에도 표시되지 않습니다. 다만 가독성이 떨어지므로, 가능하면 `<!-- -->`를 먼저 쓰는 것을 권합니다.

정리하면, 마크다운 주석은 **`<!-- -->`**가 표준적인 방법이고, HTML이 막힌 특수한 경우에만 링크 라벨 트릭을 쓰면 됩니다.
